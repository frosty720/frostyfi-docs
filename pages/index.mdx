# FrostyFi

**Visual workflow automation for AI agents + blockchain**

FrostyFi is a Web3-native platform for building AI-powered workflows with blockchain automation.

![The FrostyFi chat workspace](/screenshots/chat-workspace.png)

## Features

- **AI Chat** - Curated models routed through OpenRouter
- **Visual Workflows** - Drag-and-drop builder: LLM, HTTP, Condition, Code, EVM, Solana, Vault, Loop, and integration nodes
- **Deployable Agents** - Publish a workflow as an A2A agent with on-chain identity
- **Blockchain-native** - x402 USDC micropayments + ERC-8004 identity & reputation on Base
- **MCP Integration** - Connect remote Model Context Protocol tool servers
- **Frosty Engine** - Scheduled (cron) and webhook-triggered background workflows
- **USDC Subscriptions** - Free, Pro ($25), and Enterprise ($99) tiers on Base
